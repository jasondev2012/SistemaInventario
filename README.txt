PARA LEVANTAR LA APLICACIÓN DEBEN SEGUIR LAS SIGUIENTES INSTRUCCIONES

1.- Deben abrir la terminal. Lo pueden hacer tando click en el menú que está en la parte superior en donde dice TERMINAL y luego dan click en "New Terminal",
    esta acción abrirá un panel en la parte inferior donde deben asegurarse que estén ubicados en la carpeta SistemaInventario, y deben pegar este comando en 
    la terminal "python -m venv venv" obviar las comilla, solo copiar el contenido. Esto creará un entorno virtual que es necesario para descargar las dependecias.

2.- Una vez que se haya creado el entorno virtual, ahora debemos trabajar en él, debemos pegar en la terminal el siguiente comando "venv\Scripts\activate", 
    con esto indicamos al VISUAL STUDIO CODE que trabajaremos en un entorno virtual.

3.- Finalmente, en la misma terminal deben pegar el siguiente comando "pip install -r requirements.txt" esto instalará todas las dependecias necesarias para
    el funcionadmiento de la aplicación.
